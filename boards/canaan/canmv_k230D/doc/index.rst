Canaan CanMV-K230D
##################

Overview
********

The Canaan CanMV-K230D is a development board with a Canaan Kendryte K230
64bit RISC-V SoC.

Programming and debugging
*************************

Zephyr binary can be loaded by U-Boot command in machine mode.
